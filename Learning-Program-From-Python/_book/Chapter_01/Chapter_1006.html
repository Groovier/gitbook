附录B virtualenv

一台计算机中可以安装多个版本的Python，而使用virtualenv则可给每个版本的Python创造一个虚拟环境。下面就使用Python附带的pip^(13)^{#ch13-back}来安装virtualenv：

$pip install virtualenv

你可以为计算机中某个版本的Python创建一个虚拟空间，比如：

$virtualenv –p /usr/bin/python3.5 myenv

上面的命令中，/usr/bin/python3.5是解释器所在的位置，myenv是新建的虚拟环境的名称。下面命令可开始使用myenv这个虚拟环境：

$source myenv/bin/activate

使用下面命令可退出虚拟环境：

$deactivate
